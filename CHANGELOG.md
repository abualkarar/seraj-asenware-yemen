# سجل التغييرات

جميع التغييرات الملحوظة في هذا المشروع سيتم توثيقها في هذا الملف.

## [1.1.0] - 2024-03-08

### ✨ الإضافات
- **صفحة عن الشركة**: صفحة شاملة تحتوي على قصة الشركة والقيم والفريق والاعتمادات
- **صفحة المدونة**: مدونة تقنية مع 4 مقالات حول أنظمة كشف الحريق
- **تحسينات الملاحة**: تحديث رأس الصفحة لتضمين الصفحات الجديدة
- **ملفات التوثيق**: إضافة CONTRIBUTING.md و CHANGELOG.md و LICENSE

### 🔧 التحسينات
- تحسين هيكل المشروع
- إضافة معلومات اتصال شاملة
- تحسين تجربة المستخدم في الملاحة

### 📚 التوثيق
- تحديث README بمعلومات شاملة
- إضافة دليل المساهمة
- إضافة سجل التغييرات

## [1.0.0] - 2024-03-05

### ✨ الإضافات الأولية
- **الصفحة الرئيسية**: عرض معلومات الشركة والمميزات الرئيسية
- **صفحة المنتجات**: عرض جميع منتجات Asenware مع التصفية حسب الفئات
- **صفحة تفاصيل المنتج**: معلومات شاملة عن كل منتج مع المواصفات والتطبيقات
- **صفحة الاتصال**: نموذج تواصل متكامل مع معلومات الاتصال
- **رأس وتذييل**: ملاحة احترافية وروابط سريعة

### 🎨 التصميم
- تصميم صناعي حديث بألوان احترافية
- خطوط عالية الجودة (Roboto و Open Sans)
- صور خلفية مخصصة وعناصر بصرية ديناميكية
- تصميم متجاوب يعمل على جميع الأجهزة

### 🛠️ التقنيات
- React 19 مع TypeScript
- Tailwind CSS 4 للتصميم
- shadcn/ui للمكونات
- Wouter للملاحة
- Vite لبناء المشروع

### 📱 الميزات
- تصميم متجاوب (Responsive)
- ملاحة سهلة وسلسة
- أداء عالي
- توافقية مع جميع المتصفحات

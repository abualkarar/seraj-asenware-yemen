# السراج للتجهيزات الصناعية - موقع Asenware في اليمن

موقع إلكتروني احترافي وحديث لشركة السراج للتجهيزات الصناعية، الوكيل المعتمد لشركة Asenware في اليمن.

## 🎯 نبذة عن المشروع

موقع متخصص في عرض وتسويق أنظمة كشف الحريق الخطية والكابلات المقاومة للحريق من Asenware. يتميز الموقع بتصميم صناعي حديث وتجربة مستخدم سلسة.

## ✨ المميزات الرئيسية

- **تصميم صناعي حديث**: ألوان احترافية (أزرق داكن وأحمر جريء)
- **صفحات متكاملة**: الرئيسية، المنتجات، تفاصيل المنتج، الاتصال
- **عرض منتجات شامل**: 6 منتجات من Asenware مع تصفية حسب الفئات
- **نموذج تواصل متقدم**: للاستفسارات والطلبات
- **تصميم متجاوب**: يعمل بكفاءة على جميع الأجهزة
- **أداء عالي**: محسّن للسرعة والـ SEO

## 🛠️ التقنيات المستخدمة

- **React 19**: مكتبة JavaScript لبناء واجهات المستخدم
- **TypeScript**: لضمان سلامة الأنواع
- **Tailwind CSS 4**: إطار عمل CSS حديث
- **shadcn/ui**: مكونات واجهة مستخدم عالية الجودة
- **Wouter**: توجيه خفيف الوزن للتطبيقات أحادية الصفحة
- **Vite**: أداة بناء سريعة وحديثة

## 🚀 البدء السريع

### المتطلبات
- Node.js 18+
- pnpm أو npm

### التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/yourusername/seraj-asenware-yemen.git
cd seraj_asenware_yemen

# تثبيت الاعتماديات
pnpm install

# بدء خادم التطوير
pnpm dev
```

الموقع سيكون متاحاً على `http://localhost:3000`

## 📝 الصفحات الرئيسية

- دعوة للتواصل والطلب

- معلومات التواصل المباشرة
